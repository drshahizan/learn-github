# Exercise 3: GitHub Profile

## Learning Objective

Participants will be able to update their GitHub profile so that it looks more professional, is easy to understand and is suitable to be used as part of a digital portfolio.

## Step 1: Open Your Own Profile Page

1. Sign in to your GitHub account.
2. Click the profile icon at the top right of the GitHub page.
3. Select the `Your profile` menu.
4. Make sure your own GitHub profile page is displayed.
5. Observe the information currently shown, such as the profile picture, name, bio, links and list of repositories.

## Step 2: Open the Edit Profile Settings

1. On the profile page, click the `Edit profile` button.
2. If the button is not visible, click the profile icon and select `Settings`.
3. Select the `Public profile` section.
4. Make sure the page for updating public profile information is displayed.

## Step 3: Set a Profile Picture

1. Click the profile picture or avatar section.
2. Choose a clear and suitable picture.
3. Use a picture that is not blurry.
4. Avoid pictures that are too casual or unsuitable for academic and professional contexts.
5. Save the changes after selecting the picture.

## Step 4: Update the Display Name

1. Find the `Name` field.
2. Enter your real name or the professional name you want to display.
3. Make sure the name is spelled correctly.
4. Avoid names that are too long or difficult to read.
5. Save the changes if required.

## Step 5: Write a Short Bio

1. Find the `Bio` field.
2. Write a short description about yourself.
3. Mention your area of interest, skills or learning focus.
4. Make sure the bio is short and easy to understand.

Example bio:

```text
Computer Science Student | Interested in Web Development and Data Analytics
Information Technology Student | Mobile App, UI/UX and AI Enthusiast
Student interested in GitHub, documentation and web applications
```

## Step 6: Add Additional Information

1. Enter your location if appropriate.
2. Enter your institution or organisation if applicable.
3. Add a LinkedIn, portfolio or personal website link.
4. Make sure the links are written correctly.
5. Do not include overly personal information such as your full home address, identification card number or personal phone number.

## Step 7: Review the Profile Display

1. Return to your own profile page.
2. Check the picture, name, bio and links.
3. Make sure the information looks neat and professional.
4. Open the profile in an incognito window if you want to view it as an outside visitor.
5. Fix the information again if there are any mistakes.

## Step 8: Copy the Profile Link

1. Open your own GitHub profile page.
2. Copy the link from the address bar.
3. The link usually looks like this:

```text
https://github.com/username
```

4. Save the link for the next exercise.
5. This link will be used when participants build their digital portfolio.

## Benefits of Updating a GitHub Profile

### 1. Building a Professional Identity

A complete profile helps others recognise participants more quickly and clearly.

### 2. Making Facilitator Review Easier

A clear name, bio and links make it easier to review participants' exercises and repositories.

### 3. Supporting a Digital Portfolio

A GitHub profile can become the main page for showcasing projects, documentation and skills.

### 4. Increasing Confidence When Sharing Links

A neat profile is more suitable to share with friends, lecturers, communities or employers.

### 5. Highlighting Selected Repositories

A good profile makes it easier for participants to display their best repositories as evidence of learning and project work.

## Common Problems and How to Solve Them

| Problem | Suggested Solution |
|---|---|
| Cannot find the Edit profile button | Click the profile icon and open `Your profile`, then look for the `Edit profile` button. |
| Profile picture does not change | Refresh the page or try uploading the picture again with a smaller file size. |
| Link does not work | Make sure the link starts with `https://` and has no spelling mistakes. |
| Bio is too long | Shorten it to one or two lines containing the most important points. |
| Profile still looks empty | Add a bio, picture, location, link and pin the best repositories after the repository exercise. |
