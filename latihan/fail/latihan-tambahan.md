# Latihan Tambahan Bengkel Asas GitHub

Latihan tambahan ini disediakan untuk peserta **Bengkel Asas GitHub** yang ingin mengembangkan penggunaan GitHub kepada portfolio web, bantuan AI, editor kod dan notebook awan. Fokus utama latihan ialah menghasilkan bahan yang boleh dikongsi, disemak dan digunakan semula dalam portfolio digital peserta.

Latihan ini sesuai dilaksanakan selepas peserta memahami asas profil GitHub, repositori, README, commit dan pautan projek.

## Objektif Latihan Tambahan

Selepas melengkapkan latihan tambahan ini, peserta dapat:

1. Membina laman `github.io` menggunakan GitHub Pages.
2. Menggunakan GitHub Copilot melalui GitHub web secara bertanggungjawab.
3. Menghubungkan Microsoft Visual Studio Code dengan akaun GitHub.
4. Menggunakan Google Colab bersama GitHub untuk notebook latihan.
5. Menambah pautan hasil latihan ke dalam README projek.
6. Menghasilkan bahan portfolio digital yang lebih lengkap.

## Jadual Kandungan Latihan Tambahan

| No. | Latihan | Fokus Utama |
|---:|---|---|
| 1 | [Bina dan Terbitkan Laman github.io](fail/tambahan1.md) | Cipta laman portfolio menggunakan GitHub Pages dan pautan `github.io`. |
| 2 | [GitHub Copilot Dalam GitHub Web](fail/tambahan2.md) | Gunakan Copilot Chat untuk semakan README, cadangan Issues dan semakan kod ringkas. |
| 3 | [Hubungkan VS Code Dengan GitHub](fail/tambahan3.md) | Log masuk GitHub, clone repositori, edit fail, commit dan push melalui VS Code. |
| 4 | [Google Colab dan GitHub](fail/tambahan4.md) | Buka notebook daripada GitHub, simpan salinan, jalankan kod dan tambah pautan dalam README. |
