# Latihan 2: Follow Profil GitHub Dr Shahizan

## Objektif Latihan

Peserta dapat mengikuti profil GitHub `drshahizan` dan memahami manfaat fungsi Follow dalam GitHub untuk pembelajaran, rujukan projek dan perkembangan portfolio digital.

## Langkah 1: Log Masuk Ke GitHub

1. Buka pelayar web.
2. Pergi ke `https://github.com`.
3. Klik `Sign in`.
4. Masukkan emel atau nama pengguna GitHub.
5. Masukkan kata laluan.
6. Pastikan peserta berjaya masuk ke akaun GitHub sendiri.

## Langkah 2: Buka Profil Dr Shahizan

1. Buka tab baharu pada pelayar web.
2. Taip pautan berikut:

```text
https://github.com/drshahizan
```

3. Tekan `Enter`.
4. Pastikan halaman profil GitHub `drshahizan` dipaparkan.
5. Semak nama profil, gambar profil dan senarai repositori yang kelihatan.

## Langkah 3: Klik Butang Follow

1. Cari butang `Follow` pada halaman profil.
2. Butang ini biasanya berada berhampiran nama profil atau maklumat pengguna.
3. Klik butang `Follow`.
4. Pastikan butang berubah kepada status yang menunjukkan peserta telah mengikuti profil tersebut.
5. Jika butang masih memaparkan `Follow`, klik semula atau refresh halaman.

## Langkah 4: Semak Status Following

1. Klik ikon profil peserta di bahagian atas kanan GitHub.
2. Pilih `Your profile`.
3. Pada halaman profil sendiri, cari bahagian `following` atau senarai akaun yang diikuti.
4. Semak sama ada `drshahizan` tersenarai.
5. Jika tersenarai, proses follow telah berjaya.

## Langkah 5: Teroka Repositori Rujukan

1. Kembali ke profil `https://github.com/drshahizan`.
2. Lihat senarai repositori yang dipaparkan.
3. Buka satu repositori yang kelihatan berkaitan dengan pembelajaran atau bengkel.
4. Baca fail `README.md` repositori tersebut.
5. Perhatikan cara repositori disusun, termasuk tajuk, penerangan, fail, folder dan dokumentasi.

## Kebaikan Follow Profil GitHub

### 1. Mudah Mencari Semula Profil Rujukan

Apabila peserta follow profil `drshahizan`, peserta lebih mudah mencari semula akaun tersebut untuk melihat bahan, repositori dan contoh dokumentasi.

### 2. Mendapat Pendedahan Kepada Contoh Repositori

Peserta boleh melihat cara repositori disusun, cara README ditulis dan bagaimana bahan pembelajaran dikongsi melalui GitHub.

### 3. Belajar Melalui Pemerhatian

Peserta boleh belajar daripada struktur projek, nama fail, susunan folder, penggunaan Markdown dan cara dokumentasi disediakan.

### 4. Membina Jaringan Pembelajaran

GitHub bukan sekadar tempat menyimpan kod. Ia juga boleh digunakan untuk mengikuti pensyarah, mentor, komuniti, organisasi dan pembangun lain.

### 5. Membiasakan Diri Dengan Budaya GitHub

Fungsi Follow membantu peserta memahami bahawa GitHub mempunyai unsur komuniti. Peserta boleh mengikuti individu atau organisasi yang menghasilkan bahan berguna.

### 6. Menyokong Pembinaan Portfolio Digital

Dengan melihat contoh profil dan repositori yang baik, peserta boleh mendapat idea untuk memperbaiki profil sendiri, memilih repositori terbaik dan menulis README yang lebih kemas.

## Output Latihan

Pada akhir latihan ini, peserta perlu mempunyai:

1. Telah follow profil GitHub `drshahizan`.
2. Boleh membuka semula profil `https://github.com/drshahizan`.
3. Boleh mengenal pasti sekurang-kurangnya satu repositori rujukan.
4. Boleh menyatakan sekurang-kurangnya tiga kebaikan follow profil GitHub.
5. Mempunyai idea awal tentang cara profil dan repositori boleh dijadikan bahan pembelajaran.

## Semakan Latihan

Tandakan jika selesai:

1. Saya telah log masuk ke akaun GitHub sendiri.
2. Saya berjaya membuka profil `https://github.com/drshahizan`.
3. Saya telah klik butang `Follow`.
4. Saya telah menyemak status following.
5. Saya telah membuka sekurang-kurangnya satu repositori rujukan.
6. Saya telah membaca README dalam repositori tersebut.
7. Saya faham kebaikan follow profil GitHub.

## Masalah Biasa dan Cara Mengatasi

| Masalah | Cadangan Penyelesaian |
|---|---|
| Tidak jumpa butang Follow | Pastikan peserta telah log masuk ke akaun GitHub. |
| Profil tidak boleh dibuka | Semak semula ejaan pautan `https://github.com/drshahizan`. |
| Butang Follow tidak berubah | Refresh halaman dan semak semula status. |
| Tidak nampak senarai following | Buka profil sendiri dan cari bahagian following. |
| Repositori terlalu banyak | Pilih satu repositori sahaja untuk diteroka dahulu. |

## Nota Keselamatan

1. Follow profil tidak memberi akses kepada akaun peserta.
2. Follow tidak berkongsi kata laluan atau maklumat peribadi.
3. Peserta masih perlu berhati-hati apabila membuka pautan luar dalam mana-mana repositori.
4. Jangan salin kandungan repositori orang lain tanpa memahami, memberi kredit dan mematuhi lesen yang berkaitan.
