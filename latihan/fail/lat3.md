# Latihan 3: Profil GitHub

## Objektif Latihan

Peserta dapat mengemas kini profil GitHub supaya kelihatan lebih profesional, mudah difahami dan sesuai dijadikan sebahagian daripada portfolio digital.

## Langkah 1: Buka Halaman Profil Sendiri

1. Log masuk ke akaun GitHub.
2. Klik ikon profil di bahagian atas kanan halaman GitHub.
3. Pilih menu `Your profile`.
4. Pastikan halaman profil GitHub sendiri dipaparkan.
5. Perhatikan maklumat yang sedang kelihatan seperti gambar profil, nama, bio, pautan dan senarai repositori.

## Langkah 2: Buka Tetapan Edit Profil

1. Pada halaman profil, klik butang `Edit profile`.
2. Jika butang tersebut tidak kelihatan, klik ikon profil dan pilih `Settings`.
3. Pilih bahagian `Public profile`.
4. Pastikan halaman untuk mengemas kini maklumat profil awam dipaparkan.

## Langkah 3: Tetapkan Gambar Profil

1. Klik pada gambar profil atau bahagian avatar.
2. Pilih gambar yang jelas dan sesuai.
3. Gunakan gambar yang tidak kabur.
4. Elakkan gambar yang terlalu santai atau tidak sesuai untuk konteks akademik dan profesional.
5. Simpan perubahan selepas gambar dipilih.

## Langkah 4: Kemas Kini Nama Paparan

1. Cari ruangan `Name`.
2. Masukkan nama sebenar atau nama profesional yang ingin dipaparkan.
3. Pastikan ejaan nama betul.
4. Elakkan nama yang terlalu panjang atau sukar dibaca.
5. Simpan perubahan jika diminta.

## Langkah 5: Tulis Bio Ringkas

1. Cari ruangan `Bio`.
2. Tulis penerangan ringkas tentang diri.
3. Nyatakan bidang minat, kemahiran atau fokus pembelajaran.
4. Pastikan bio ringkas dan mudah difahami.

Contoh bio:

```text
Pelajar Sains Komputer | Minat Web Development dan Data Analytics
Computer Science Student | Mobile App, UI/UX and AI Enthusiast
Pelajar Teknologi Maklumat | Berminat dalam GitHub, dokumentasi dan aplikasi web
```

## Langkah 6: Tambah Maklumat Tambahan

1. Masukkan lokasi jika sesuai.
2. Masukkan institusi atau organisasi jika ada.
3. Tambah pautan LinkedIn, portfolio atau laman peribadi.
4. Pastikan pautan ditulis dengan betul.
5. Jangan masukkan maklumat yang terlalu peribadi seperti alamat rumah penuh, nombor kad pengenalan atau nombor telefon peribadi.

## Langkah 7: Semak Paparan Profil

1. Kembali ke halaman profil sendiri.
2. Semak gambar, nama, bio dan pautan.
3. Pastikan maklumat kelihatan kemas dan profesional.
4. Buka profil dalam tetingkap incognito jika mahu melihat paparan sebagai pelawat luar.
5. Baiki semula maklumat jika ada kesilapan.

## Langkah 8: Salin Pautan Profil

1. Buka halaman profil GitHub sendiri.
2. Salin pautan pada address bar.
3. Pautan biasanya berbentuk seperti berikut:

```text
https://github.com/nama-pengguna
```

4. Simpan pautan tersebut untuk latihan seterusnya.
5. Pautan ini akan digunakan apabila peserta membina portfolio digital.

## Kebaikan Mengemas Kini Profil GitHub

### 1. Membina Identiti Profesional

Profil yang lengkap membantu orang lain mengenali peserta dengan lebih cepat dan jelas.

### 2. Memudahkan Semakan Fasilitator

Nama, bio dan pautan yang jelas memudahkan semakan latihan dan repositori peserta.

### 3. Menyokong Portfolio Digital

Profil GitHub boleh menjadi halaman utama untuk mempamerkan projek, dokumentasi dan kemahiran.

### 4. Meningkatkan Keyakinan Apabila Berkongsi Pautan

Profil yang kemas lebih sesuai dikongsi kepada rakan, pensyarah, komuniti atau majikan.

### 5. Menonjolkan Repositori Pilihan

Profil yang baik memudahkan peserta memaparkan repositori terbaik sebagai bukti pembelajaran dan kerja projek.

## Masalah Biasa dan Cara Mengatasi

| Masalah | Cadangan Penyelesaian |
|---|---|
| Tidak jumpa butang Edit profile | Klik ikon profil dan buka `Your profile`, kemudian cari butang `Edit profile`. |
| Gambar profil tidak berubah | Refresh halaman atau cuba muat naik semula gambar dengan saiz fail yang lebih kecil. |
| Pautan tidak berfungsi | Pastikan pautan bermula dengan `https://` dan tiada kesalahan ejaan. |
| Bio terlalu panjang | Pendekkan kepada satu atau dua baris yang paling penting. |
| Profil masih nampak kosong | Tambah bio, gambar, lokasi, pautan dan pin repositori terbaik selepas latihan repositori. |
