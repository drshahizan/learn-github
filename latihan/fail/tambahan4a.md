# Additional Exercise 4: Google Colab and GitHub

This exercise guides participants in using **Google Colab** with **GitHub**. The focus is on opening a notebook from GitHub, making a copy of the notebook, running simple code and saving the notebook link as part of project documentation.

## Learning Objectives

After completing this exercise, participants will be able to:

1. Open Google Colab through a web browser.
2. Access a notebook from GitHub.
3. Run simple code in a notebook.
4. Save a copy of the notebook to Google Drive.
5. Add a Google Colab link to the project README.

## Step 1: Open Google Colab

1. Open a web browser.
2. Go to the following link:

```text
https://colab.research.google.com
```

3. Sign in using your Google account.
4. Make sure the Google Colab page opens successfully.
5. If asked to choose an account, select the account that will be used for the exercise.

## Step 2: Open the GitHub Tab in Colab

1. In the Colab start window, select the **GitHub** tab.
2. If the window does not appear, click the **File** menu.
3. Select **Open notebook**.
4. Select the **GitHub** tab.
5. Make sure the GitHub search field is displayed.

## Step 3: Search for the Training Repository

1. In the GitHub search field, enter the training repository link.
2. Use the following link as an example:

```text
https://github.com/drshahizan/learn-github
```

3. Press `Enter`.
4. If there are notebooks in the repository, the list of notebook files will be displayed.
5. Select the notebook you want to open.

## Step 4: Open a Notebook from GitHub

1. Click the name of the notebook file you want to use.
2. Make sure the notebook opens in Google Colab.
3. Check the notebook title and cell content.
4. Do not change the original file without making a copy.
5. Use the notebook as exercise material or a reference.

## Step 5: Save a Copy of the Notebook

1. Click the **File** menu.
2. Select **Save a copy in Drive**.
3. Google Colab will open a new copy of the notebook.
4. This copy is saved in your Google Drive.
5. Rename the notebook so that it is clearer.
6. Example file name:

```text
colab-github-exercise-your-name.ipynb
```

## Step 6: Run Simple Code

1. Add a new code cell in the notebook.
2. Enter the following simple code:

```python
name = "Your Name"
github_id = "your-githubid"

print("Name:", name)
print("GitHub ID:", github_id)
print("I am learning to connect Google Colab with GitHub.")
```

3. Click the run button on the cell.
4. Make sure the output is displayed correctly.
5. Change `Your Name` and `your-githubid` to your actual information.

## Step 7: Add a Markdown Cell

1. Add one text or Markdown cell.
2. Enter brief information like the following:

```markdown
# Google Colab and GitHub Exercise

Name: Your Name

GitHub ID: your-githubid

Exercise purpose: Open, run and share a Google Colab notebook related to GitHub.
```

3. Make sure the text is displayed neatly.
4. Use suitable headings and line spacing.

## Step 8: Get the Notebook Link

1. Click the **Share** button at the top right of Google Colab.
2. Set access according to the facilitator's instructions.
3. If the exercise needs to be shared, choose a suitable link setting.
4. Copy the notebook link.
5. Save the link to add it to the project README.

## Step 9: Add the Colab Link in README

1. Open your training repository in GitHub.
2. Open the `README.md` file.
3. Click the pencil icon to edit the file.
4. Add a new section like the following:

```markdown
## Google Colab

Training notebook link:

[Open Google Colab Notebook](your-notebook-link)
```

5. Replace `your-notebook-link` with your actual notebook link.
6. Check the Markdown display before committing.

## Step 10: Commit Changes in GitHub

1. Scroll to the bottom of the README edit page.
2. Write a clear commit message.
3. Example commit message:

```text
Add Google Colab link in README
```

4. Click **Commit changes**.
5. Check the README again after the commit.
6. Make sure the Google Colab link can be opened.

## Common Problems and How to Solve Them

| Problem | Action |
|---|---|
| Cannot open Google Colab | Check the internet connection and Google account. |
| GitHub tab does not display files | Make sure the repository link is correct and the repository can be accessed. |
| Notebook cannot be saved | Make sure the Google Drive account has enough storage. |
| Code does not run | Check the code syntax again and run the cells one by one. |
| Colab link cannot be opened | Check the notebook sharing settings. |
