# Basic GitHub Workshop Training

This training module is prepared for participants of the **Basic GitHub Workshop**. The training focuses on the practical use of GitHub through a web browser and GitHub Desktop. It emphasises the development of a professional profile, project repositories, README documentation, task management, digital portfolios, security and academic ethics.

## Learning Objectives

After completing the training, participants will be able to:

- Build a professional GitHub profile as a digital portfolio.
- Create a well-organised project repository with complete README documentation using basic Markdown.
- Manage and publish project materials through GitHub web and GitHub Desktop while applying appropriate security and academic ethics practices.

## Training Table of Contents

### Part A: Account and Profile Preparation

| No. | Exercise | Main Focus |
|---:|---|---|
| 1. | [Create a GitHub Account](fail/lat1a.md) | Register an account, verify the email address and save the profile link. |
| 2. | [Follow Dr Shahizan's GitHub Profile](fail/lat2a.md) | Follow a reference profile and understand the benefits of the Follow feature. |
| 3. | [GitHub Profile](fail/lat3a.md) | Update the profile picture, name, bio, links and profile display. |

### Part B: Repositories and Documentation

| No. | Exercise | Main Focus |
|---:|---|---|
| 4. | [Create Your First Project Repository](fail/lat4a.md) | Create a repository, enable README and add initial information. |
| 5. | [Mad_club Repository](fail/lat5a.md) | Write a complete project README for Mad_club. |
| 6. | [Using Markdown in a README File](fail/lat6a.md) | Headings, lists, links, images, tables and simple code. |
| 7. | [Upload Project Files](fail/lat7a.md) | Add files through GitHub web, organise folders and commit changes. |
| 8. | [Images in README](fail/lat8a.md) | Upload screenshots, display images and add demo links. |

### Part C: GitHub Desktop

| No. | Exercise | Main Focus |
|---:|---|---|
| 9. | [Install, Sign In and Set Up GitHub Desktop](fail/lat9a.md) | Download GitHub Desktop, sign in to a GitHub account and check the active account. |
| 10. | [Clone a Repository to the Computer](fail/lat10a.md) | Clone the training repository to the computer and check project files locally. |
| 11. | [Edit README, Commit and Push](fail/lat11a.md) | Edit the README file, commit changes and push to GitHub. |
