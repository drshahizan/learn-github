# ✅ Pre-Workshop Preparation

Please complete the following two steps **before attending the workshop**. This will allow you to jump straight into the hands-on session without any delays.

## Step 1 — Create a GitHub Account

1. Go to **[github.com](https://github.com)** and click **Sign up**
2. Enter your email address, password and username
   > 💡 Use a professional username — e.g. `shahizan`, `ahmad-rizal`, not `coolboy99`
3. Select the **Free** plan
4. Check your email and click the verification link from GitHub
5. Log in and complete your basic profile:
   - A clear profile picture
   - Your full name
   - A short bio (e.g. *Student at UTM | Interested in data science*)

## Step 2 — Download GitHub Desktop

1. Go to **[desktop.github.com](https://desktop.github.com)**
2. Click **Download for Windows** (or macOS if applicable)
3. Install the application and sign in using your newly created GitHub account
4. Confirm the installation was successful — the GitHub Desktop home screen should appear

## Final Checklist

Before coming to class, make sure you have:

- [ ] A GitHub account verified via email
- [ ] Successfully logged in at **github.com**
- [ ] GitHub Desktop installed and signed in
- [ ] A working laptop with internet access
