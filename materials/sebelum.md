# ✅ Persediaan Sebelum Bengkel

Sila lengkapkan dua langkah berikut **sebelum hadir ke bengkel**. Ini akan membantu anda memulakan sesi praktikal dengan segera tanpa perlu menunggu.

## Langkah 1 — Daftar Akaun GitHub

1. Pergi ke **[github.com](https://github.com)** dan klik **Sign up**
2. Masukkan alamat emel, kata laluan dan nama pengguna
   > 💡 Gunakan nama pengguna yang profesional — contoh: `shahizan`, `ahmad-rizal`, bukan `coolboy99`
3. Pilih pelan **Free**
4. Semak emel dan klik pautan pengesahan dari GitHub
5. Log masuk dan lengkapkan profil asas:
   - Gambar profil yang jelas
   - Nama penuh
   - Bio ringkas (contoh: *Student at UTM | Interested in data science*)



## Langkah 2 — Muat Turun GitHub Desktop

1. Pergi ke **[desktop.github.com](https://desktop.github.com)**
2. Klik **Download for Windows** (atau macOS jika berkenaan)
3. Pasang aplikasi dan log masuk menggunakan akaun GitHub yang baru didaftarkan
4. Sahkan pemasangan berjaya — skrin utama GitHub Desktop sepatutnya muncul



## Semakan Akhir

Sebelum masuk ke kelas, pastikan anda sudah:

- [ ] Mempunyai akaun GitHub yang telah disahkan melalui emel
- [ ] Berjaya log masuk di **github.com**
- [ ] GitHub Desktop dipasang dan log masuk berjaya
- [ ] Membawa laptop yang berfungsi dengan capaian internet
