## Introduction

This workshop introduces GitHub as a platform for storing project materials, producing documentation, building a digital portfolio and managing team collaboration.

The workshop focuses on the practical use of GitHub through the web interface, GitHub Desktop, Visual Studio Code and other supporting applications. Participants do not need prior experience using Git commands through the command line.

## Workshop Objectives

After completing this workshop, participants will be able to:

1. Build a GitHub profile and project repository suitable for a digital portfolio.
2. Produce project documentation using README, Markdown, images and demo links.
3. Use GitHub for team collaboration through Issues, Projects and Pull Requests.

## Workshop Outcomes

At the end of the workshop, participants are expected to have:

- An active GitHub account.
- A more professional GitHub profile.
- At least one project repository.
- A neat and complete project README.
- Experience using GitHub Desktop.
- Experience collaborating in a group repository.

## For Participants

[Before the workshop](https://github.com/drshahizan/learn-github/blob/main/materials/before.md):

- Prepare an active email address.
- Make sure you can sign in to your email account.
- Bring a laptop if available.
- Make sure the computer can access GitHub through a web browser.

During the workshop:

- Follow the exercises in sequence.
- Save your GitHub profile link.
- Save your project repository link.
- Use neat file and folder names.
- Ask the facilitator if you face technical issues.

After the workshop:

- Update your GitHub profile.
- Update your project README.
- Check your project and demo links again.
- Share your profile or repository link as a digital portfolio.
