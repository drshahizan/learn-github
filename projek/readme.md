# Mini Projek GitHub

Mini projek ini disediakan untuk peserta **Bengkel Asas GitHub** sebagai latihan amali berkumpulan. Fokus utama mini projek ialah kolaborasi antara ahli kumpulan dalam satu repositori melalui collaborator, Issues, GitHub Projects, fail sumbangan ahli, Pull Request, review dan README projek.

Mini projek ini sesuai dilaksanakan dalam masa ringkas, sekitar 1 jam, dengan satu kumpulan terdiri daripada 4 orang ahli. Setiap kumpulan perlu menghasilkan satu repositori projek yang menunjukkan bukti kerja berpasukan.

## Objektif Mini Projek

Selepas melengkapkan mini projek, peserta dapat:

1. Membina repositori projek kumpulan dengan ahli, fail sumbangan individu dan README yang dikemas kini secara profesional.
2. Mengurus pembahagian tugasan menggunakan Issues dan GitHub Projects Board.
3. Menunjukkan bukti kolaborasi pasukan melalui commit, Issues, Projects dan Pull Request.

## Jadual Kandungan Mini Projek

### Bahagian A: Persediaan Repositori dan Collaborator

| No. | Latihan | Fokus Utama |
|---:|---|---|
| 1 | [Cipta Repositori Projek Kumpulan](fail/mp1.md) | Ketua kumpulan mencipta repositori projek, aktifkan README dan kongsi pautan. |
| 2 | [Tambah Ahli Sebagai Collaborator](fail/mp2.md) | Ketua menjemput 3 ahli kumpulan dan semua ahli menerima jemputan. |

### Bahagian B: Agihan Tugasan dan Pengurusan Projek

| No. | Latihan | Fokus Utama |
|---:|---|---|
| 3 | [Cipta Issues Untuk Agihan Tugasan](fail/mp3.md) | Cipta sekurang-kurangnya 4 Issues, satu tugasan untuk setiap ahli. |
| 4 | [Cipta GitHub Projects Board](fail/mp4.md) | Susun Issues dalam board dengan lajur To Do, In Progress, Review dan Done. |

### Bahagian C: Sumbangan Ahli dan Semakan Pasukan

| No. | Latihan | Fokus Utama |
|---:|---|---|
| 5 | [Setiap Ahli Edit Fail Sendiri](fail/mp5.md) | Setiap ahli tambah fail Markdown sendiri dalam folder `ahli` dan commit perubahan. |
| 6 | [Pull Request dan Semakan Pasukan](fail/mp6.md) | Buka Pull Request, ahli lain review, merge perubahan dan tutup Issue. |

### Bahagian D: Dokumentasi Akhir Projek

| No. | Latihan | Fokus Utama |
|---:|---|---|
| 7 | [Kemas kini README Projek](fail/mp7.md) | Lengkapkan README utama dengan nama projek, ahli, peranan, pautan fail dan status. |
